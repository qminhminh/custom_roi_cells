═══════════════════════════════════════════════════════

CÁCH 1: ANDROID STUDIO (KHUYÊN DÙNG) ⭐⭐⭐
─────────────────────────────────────────────
1. Mở Android Studio
2. File → Open → Chọn thư mục "example"
3. Chọn device/emulator ở thanh trên
4. Nhấn nút Run (▶) màu xanh
5. Đợi app chạy (1-2 phút)
6. XEM GIAO DIỆN!

─────────────────────────────────────────────

CÁCH 2: SCRIPT TỰ ĐỘNG (WINDOWS)
─────────────────────────────────────────────
1. Double-click file: example/CHAY_NHANH.bat
2. Đợi script chạy
3. App sẽ tự động mở

─────────────────────────────────────────────

CÁCH 3: TERMINAL
─────────────────────────────────────────────
cd example
flutter pub get
flutter run

─────────────────────────────────────────────

GIAO DIỆN SẼ HIỂN THỊ:
─────────────────────────────────────────────
✅ Header màu xanh với hướng dẫn
✅ Grid cells 15x15 (225 cells)
✅ Tap để chọn cells (màu đỏ)
✅ Drag để chọn nhiều cells
✅ Nút Save, Delete, Clear
✅ Hiển thị mảng index: [0,1,2,3,...]

─────────────────────────────────────────────

CHỨC NĂNG CẦN TEST:
─────────────────────────────────────────────
□ Tap vào cell → Chọn (màu đỏ)
□ Tap lại → Bỏ chọn
□ Drag → Chọn nhiều cells
□ Nhấn Save → Lưu mảng index
□ Nhấn Delete → Xóa selection
□ Nhấn Clear → Xóa tất cả

─────────────────────────────────────────────

NẾU GẶP LỖI:
─────────────────────────────────────────────
→ Mở Android Studio và chạy từ đó
→ Đảm bảo đã có emulator hoặc điện thoại
→ Xem file HUONG_DAN_CHAY_APP.md để biết thêm

─────────────────────────────────────────────

FILE HƯỚNG DẪN CHI TIẾT:
─────────────────────────────────────────────
- CHAY_APP_NHANH.md (Hướng dẫn nhanh)
- XEM_GIAO_DIEN.md (Mô tả giao diện)
- HUONG_DAN_CHAY_APP.md (Hướng dẫn đầy đủ)
